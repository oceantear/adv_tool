# adv_gep.py
Dynamic change Unix time stamp to YYYY-MM-DD

Tool Command
======



-h, tool help
======

<pre>adv_grep -h </pre>

--version, show tool version
======

-f, which file want to be processed
======

<pre>adv_grep -f filename</pre>


-t, time filter
======

<pre>adv_grep -t YYYY-MM-DD</pre>
