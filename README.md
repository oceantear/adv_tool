# adv_tool
Dynamic change Unix time stamp to YYYY-MM-DD

Tool Command
======



-h, 
======
tool help
<pre>adv_grep -h </pre>

-f, 
======
which file want to be processed
<pre>adv_grep -f filename</pre>


-t, time filter
======
Only show the day you are interisting

<pre>adv_grep -t YYYY-MM-DD</pre>
